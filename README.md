# 🛒 E-commerce API - Node.js

A scalable and modern RESTful E-commerce API built with Node.js, Express, and MongoDB.  
This project provides authentication, product management, cart handling, order processing, and role-based access control for a complete backend e-commerce system.

---

# 🚀 Features

- 🔐 JWT Authentication & Authorization
- 👤 User Registration & Login
- 🛍 Product CRUD Operations
- 📦 Category Management
- 🛒 Shopping Cart System
- ❤️ Wishlist Support
- 📄 Order Management
- 💳 Payment Ready Structure
- 🛡 Role-Based Access Control (Admin/User)
- 🔎 Search & Filtering
- 📁 Clean Folder Structure
- 🌍 RESTful API Architecture
- ⚡ Error Handling Middleware
- 🔒 Secure Password Hashing
- 📑 Environment Variables Support

---

# 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- dotenv
